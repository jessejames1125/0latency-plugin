# 0latency plugin (`0l`)

Live meeting capture -> frame-grounded GitHub issues, automatically.
Install: `/plugin marketplace add jessejames1125/0latency-plugin` then `/plugin install 0l`.
Setup: `/0l:setup` · Run: `/0l:start <title>` · End: `/0l:finish`
